# Hero Hunter Road Quest

A live race-competition web app built for HeroMotorCorp.

Some of its features include:
- Leaderbord with live updates
- Map with live location of bike riders sharing their GPS location. Implemented with Google maps API
- Admin dashboard
- Chat Application
- Image Upload
- Complete CRUD functionality
- User/Admin Authentication & Authorization

[View Live Web application](https://herosuperoga.com)
